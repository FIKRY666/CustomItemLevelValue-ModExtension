# CustomItemLevelValue - Mod扩展框架

《逃离鸭科夫》的Mod扩展框架，为其他Mod提供无缝集成显示功能。支持五段式显示位置、富文本、颜色跟随等特性。

## ✨ 特性

- **五段式显示**：在物品信息面板的5个预设位置显示扩展内容
- **富文本支持**：支持游戏内所有富文本标签（颜色、大小、图标等）
- **颜色跟随**：自动匹配主Mod的颜色模式（白灰配色、稀有度配色等）
- **字体调节**：跟随主Mod的字体大小设置
- **零性能影响**：实时扫描，无缓存开销
- **简单集成**：只需几行代码即可集成

## 📁 显示位置

框架提供5个固定显示位置：

| 位置 | 显示顺序 | 推荐内容 |
|------|----------|----------|
| **Top1** | 稀有度后，价值前 | 状态信息、评分等级 |
| **Top2** | 价值后，属性前 | 需求趋势、市场信息 |
| **Top3** | 属性后，功能信息前 | 特殊效果、套装信息 |
| **Bottom1** | 描述后，耐久度前 | 背景故事、来源信息 |
| **Bottom2** | 耐久度前 | 使用提示、维护信息 |

## 🚀 快速集成

### 1. 添加字段到物品

```csharp
// 在物品悬停时添加扩展字段
private void OnItemHovered(ItemHoveringUI ui, Item item)
{
    // 使用标准命名格式：{Mod前缀}_{位置}_{字段名}
    item.Variables.SetString("MyMod_Top1_状态", "[c=#55FF55]✓ 可用[/c]");
    item.Variables.SetString("MyMod_Top2_评分", "评分: [c=#FFD700]★★★★☆[/c]");
}
